# AGENTS.md

This repository keeps a **single source of truth** for agent and contributor
guidance in [`CLAUDE.md`](CLAUDE.md). Read it before working in this repo — it
covers the project overview, build & test commands, workspace architecture, key
types, the core execution pipeline, and the working rules.

`AGENTS.md` exists so tools that look for it are routed to the same
instructions; it intentionally does not duplicate that content to avoid the two
files drifting apart.
